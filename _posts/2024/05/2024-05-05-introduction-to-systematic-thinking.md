# 问题

## 1. 世界的复杂性

认识世界的第一步，是承认我们自身的局限性。

我们所掌握的知识，相比于世界的浩瀚与复杂，实在微不足道。

## 2. 机械论与机械力学

以牛顿对太阳系天体运动的解释为例，尽管太阳系中存在着无数天体和未成形物质，但早期的行星分析却忽略了绝大多数物质，这在今天看来是难以想象的。

正如松果体之于大脑、DNA之于细胞、蜂王之于蜂群，如果仅从质量的角度考虑，它们在整体中都显得微小。然而，当我们深入探究人体的奥秘、细胞生物学原理以及蜂群的社会组织时，会发现它们的作用是不可或缺的。

因此，我们可以说，力学是一门研究特定系统的科学，这些系统能够很好地被力学近似原理所描述。

## 3. 计算的平方律

当问题规模增大时，计算量将如何增长？

计算的平方律指出：描述 N 个物体之间的关系，所需的关联方程数量将以 2 的 n 次方形式增长。

## 4. 科学的简化与简化的科学

牛顿在研究太阳系天体运动时，采取了大量的简化措施：

1.  他只考虑了万有引力，忽略了地磁效应、静电作用等其他因素。
2.  他将万有引力方程 F=GMm/r² 设定为两个物体之间的作用力在任何时候都与第三者无关。
3.  由于太阳的质量远大于其他行星质量的总和，因此他忽略了行星之间的相互作用力。

通过这些简化，原本需要耗费大量时间和金钱才能完成的计算，变得可以通过人脑进行。

求解一个包含 n 个方程的系统需要 n² 次计算，而计算 n 个仅含 1 个方程的独立系统，则只需要 n 次计算。

正是由于计算的平方律的存在，以及世界本身的复杂性，我们才需要进行简化。
## 5. 统计力学与大数定律

在考虑空气的特性时，由于计算平方律的限制，我们往往只能关注其平均特性。

大数定律指出，观测样本的数量越多，观测值就越接近于预测的平均值。

薛定谔提出了 N 的平方根定律，该定律表明：

偏离的量级约为 √n，相对误差的数量级为 1/√n，其中 n 为共同体现这些定律的分子数。

随机性是保证统计计算结果正确的关键特性。

## 6. 中数定律

通过以上对机械力学和统计学的分析，我们可以总结出一张图表（*此处应插入图表*）。

图表中，区域 3 代表着有序的复杂性，这正是我们所要关注的“中数系统”。

计算的平方律和 N 的平方根定律，都不适用于中数系统。将这两种定律结合起来，便得到了中数定律。

中数定律指出：对于中数系统而言，可以预见的是，它与任何理论之间都或多或少地存在着较大的波动、不规则性或偏差。

中数定律的重要性，不在于其预测能力，而在于其广泛的适用范围。严格意义上的机械学系统和统计学系统都十分罕见，我们周围充斥着各种各样的中数系统。

中数定律可以被理解为墨菲定律——凡是有可能发生的，终将发生。

科学在特定领域取得了巨大成功，但却无法有效应用于中数系统。

因此，我们需要对中数系统进行一定的“压制”。

科学技术在分解和细化的过程中取得了前所未有的成功，但同时也带来了诸多问题。我们往往忽略了，在分析和求均值之外，还会产生哪些潜在的后果。

而系统化思维，正是一种关注于寻找解决复杂问题途径的思维方式。
系统化思维，关注的是寻找解决复杂问题的途径。

# 方法

## 1. 有机体、类推与活机论

系统学研究试图通过发现一般规律，来促进对中数系统的思考，并强调规律的可用性。

有机论则试图将与生命系统相关的知识，类推到其他系统，从而获得处理复杂问题的方法。

然而，这种做法的问题在于，我们对生命系统本身的认知还不够深入，且类推过程可能不够严谨。

但需要明确的是，任何模型都是用我们已知的知识，去理解未知的领域。

科学尚未能成功地将一切现象都归结为物理和化学原理，这便为有机论留下了发挥空间。

当然，科学是否能够解释一切，本身就是一个纯粹的哲学问题。

从根本上说，科学也是一种信仰。在科学无法解释的领域，有机思维能够为我们提供有益的帮助。

## 2. 科学家及其分类

在此，我们关注的是思维的运作方式，而非评判其对错。

一些科学家之所以能在不同领域取得成功，并非因为他们改变了自己的思维模式，而是因为他们将原有的思维模式迁移到了新的领域。

要成为一名出色的通才，就应该对任何事物都保持开放的心态，不预设任何先决信念。

在罗素看来，信念就是在没有任何证据的情况下，仍然相信某种事物。

## 3. 系统论信念的主旨

但是，人无法脱离信念而生存。如果没有信念，我们甚至无法迈出下一步，因为我们无法确定脚下的地面是否能够支撑我们的体重。

一般系统论的主要信念是：经验世界自身的秩序，具有一种被称为“二阶序”的秩序。

发现一般系统规律的主要方法是归纳。

由于归纳法并非普遍有效，因此我们可能会犯错，但它能够加快我们前进的步伐。

## 4. 系统论规律的本质

*   **定律守恒定律**——当发现事实与既有定律不符时，可以选择拒绝接受该事实，或者修改定律的定义，但绝不能抛弃定律本身。
*   **喜悦特性定律**——每一条一般定律至少应适用于两种情况。
*   **沮丧滥用定律**——任何一般规律至少应存在两个例外情况。换言之，如果你从未犯错，那就意味着你什么都没有尝试。
*   **组合定律**——整体大于部分之和。
*   **分解定律**——部分大于整体之局部。

面对中数系统，我们无法依赖上述定律获得精确的估计。但是，它们可以帮助我们避免在通往正确估计的道路上犯下重大错误。

## 5. 系统化思维的类型

*   **促进思维过程**——确定思维方式，提出尖锐的问题。
*   **研究特殊系统**——关注真实的需求和真正的目的。
*   **创造新的定律、改造旧的定律**——发明并加以运用。

系统学正在不断发展壮大，我们是否可以避免建造“庞大机器”，转而构建更加友好的体系？

我还想做最后的尝试，那就是将系统学带回到它本应服务的普通大众身边，这是一种竭尽全力的努力。

# 系统与幻觉

## 1. 一个系统就是对世界的一种看法

不同的人可能会看到不同的事物，例如，一个女巫、一个少女，或者什么也看不到。

最值得反思的错误观点是，有些人试图探究画作的“真实”含义，他们意识到了自我中心论的局限性，并认为自己站在更高的层面进行思考。

然而，认为真理可以独立于观察者而存在的观点，本身就是一种最大的自我中心论。如果真的存在这样的真理，又有谁能够发现它呢？

*   如果我是一颗行星，我又是如何被太阳的巨大质量所吸引的？
*   如果我是一只牡蛎，我会因为一粒沙子而感到烦躁吗？
*   如果我是一只青蛙，我会轻易地被影子吓到吗？
*   如果我是大自然，我会选择说谎吗？

以上问题都在某种程度上阻碍了科学的发展。

只有坚信外部世界的真实存在，我们才能在科学的道路上不断前进。

“外部世界独立于人的感知而存在”——这一信念构成了所有科学的基础（爱因斯坦）。

但这本身也是一种启发式思维方式。

力学家无法明确指出哪些系统符合力学规律，数学家也无法准确界定数学的应用范围。

这就是“香蕉原理”——启发式思维方式无法确定其适用的边界。

我们抱着启发式思维一路前行，并对此深信不疑。但我们需要意识到，我们所坚信的，可能仅仅是建立在假设的基础之上的一种幻觉，尽管这种幻觉是如此强烈，以至于我们难以忘怀。

## 2. 绝对思维与相对思维

对于非人造系统，我们往往不清楚其产生的真正目的，因此也难以形成绝对思维。例如，我们不能认为大自然的存在就是为了服务于人类。

对于人造系统，我们通常清楚其创建的目的，因此容易形成绝对思维。例如，汽车工厂在生产汽车的同时，也会产生垃圾，但我们往往只关注其生产汽车的功能。

## 3. 系统是一个集合

当我们采用数学集合的方式来描述系统时，往往会忽略观察者对集合的影响。

观察者实际上决定了如何选择集合中的元素。

选择元素的方法主要有：穷举法、典型元素法和规则表示法。

由于名称的模糊性和不确定性，穷举法这种看似基础的操作，实际上也存在一定的风险。

更糟糕的是，用推导法代替穷举法，尤其是在使用典型元素来表示集合时，更容易出现问题。

规则表示法的问题在于，我们很难清晰地表达规则的所有细节。

## 4. 观察者与观察结果

从观察者的角度来看，使用笛卡尔积可能是一种更为严谨的方式。

它从两个方面来理解观察者：其能够观察的类型（广度），以及在每种类型中，其能够选择的结果范围（深度）。

例如，{A类型×B类型}，其中 A:{a1,a2,a3}，B：{b1,b2,b3,b4}。

由于笛卡尔积可能会产生组合错误，因此集合中可能会出现无法观察到（即不存在）的元素。

但是，只要观察的广度和深度被恰当地定义，那么至少可以确保在观察集合中，不会排除任何观察者能够观察到的结果。

## 5. 无关法则

无关法则指出：定律不会因为其所选择的特定符号而发生改变。

一旦采用这种集合方法，再加上符号的无关法则，就能够有效减少名称的模糊性，从而对不同观察者的结果进行一致性比较。

如果对于 B 中的每一个元素，A 中都不会出现两个不同的元素，那么我们可以认为 A 和 B 是一致的。

对于同样的事物，如果 A 比 B 的观察分类更细致、分辨率更高，那么作为观察者，A 就比 B 更具优势。

而理想化的“超级观察者”必须具有超越现有任何观察者的视角——在极端情况下，它必须能够看到包含所有观察者观察结果的各种可能组合的乘积空间。

然而，这种笛卡尔积的乘积空间的组合数量，会随着观察人数的增长而呈指数形式增长，因此，对于中等以上复杂程度的问题，我们很难设想“超级观察者”能够发挥有效作用。

# 对观察结果的解释

## 1. 状态

对于同一个黑箱音乐盒的组合状态，从超级观察者、有听力障碍的物理学家以及音乐盒发明者的角度来看，会得出截然不同的观察结论。

## 2. 眼-脑定律

物理学家可能会忽略某个音阶，发明者可能会忽略灯光闪烁（因为他知道这没有实际意义），而超级观察者虽然观察到了所有的细节，却容易陷入只见树木不见森林的困境。

在某种程度上，脑力和眼力是可以相互弥补的。但是，“脑”和“眼”之间的平衡不能过度偏向任何一方（过于精确的测量结果反而会给结论的建立造成障碍）。因此，科学的任务就是要找到两者之间的最佳平衡点。

## 3. 广义热力学定律

我们应该承认，任何所谓的“客观”事物，都是由两个方面的要素构成的：物质本原和观察者的精神倾向。忽略后者同样容易使我们陷入主观的泥潭。

*   **广义热力学定律（第一种形式）**——在没有特殊限制的情况下，出现概率较大的状态，要比出现概率较小的状态更容易被观察到。
*   **广义热力学定律（第二种形式）**——我们频繁看到的事物，往往也是频繁发生的，原因在于：

    1.  存在满足某种状态所需的物理原因。
    2.  存在某种精神上的原因。

从“人”的角度考虑，我们不能试图研究事物的全部情况，否则所有事物都将呈现出细微的差别，进而成为特例，科学分类和规律也将无从谈起。

**结合定律**——在研究任何事物时，都不能试图研究其所有的情况。

#### 4. 功能符号与简化思想

因此，我们通常会从自身的角度出发，选择事物的若干重要特征，并建立函数：y=f(a,b,x)。但由于观察的局限性，我们可能会忽略某些具有重要影响的特征。另一方面，对于影响因子的测量也无法无限精确下去。当达到某个微观层面时，因子 a 与 b 甚至会此消彼长，从而相互矛盾。因此，通过观察，我们无法获得全部的信息。

#### 5. 不完全与过于完全

#### 6. 广义互补性原理

**广义互补定律**——任何两种观点都是互补的。

# 对观察结果的分解

人脑的能力是有限的，而人类智力中唯一看似无限的能力，或许就是自我欺骗的能力——尤其是在面对人脑是否无限这个问题时。

在面对不熟悉且复杂的对象时，我们通常会先试图获取一种“全面”的观点，力求包含所有的现象。

然后，我们会尝试获得一种“最小”的观点，合并那些不必要区分的状态，以减轻观察的负担。

然而，这两种方法都存在其固有的局限性。

那么，我们再来看看常用的第三种方法：由于人脑的局限性，我们转而获取一种“独立”的观点，即将观察到的状态分解成若干不相关的部分，以此来减轻脑力的负担。

**差别法则**——定律不应该依赖于特定的符号表示，但事实却往往与此相反。

## 1. 科学的隐喻

**经验公理**——未来会像过去一样，因为在过去，未来也总是像过去一样。

面对同样的系统，不同的人会以不同的方式将其分解为不同的子系统集合。这主要是因为每个人都受到自身分解世界的固有模式的影响。

通过这种类似于“比喻”的方式，子系统被比喻成不同的事物。

## 2. 事物与边界

子系统之间原本渐变且相互交错的边界，往往会被过度简化，从而割裂整个系统的有机性。

## 3. 性质与不变法则

在分解过程中，我们往往会根据事物的性质来划分子系统。

而对于观察者而言，性质本身就具有精神上的作用。

我们可能会认为某些性质比其他性质更为“自然”，但这仅仅表明我们更习惯于以那样的方式进行观察。

因此，在某种分解过程中，一些部分之所以能够成为子系统并保持不变，是因为它们被观察者认为是“有价值”的。

反之，那些成为边界的部分，则往往被认为是“不重要”的。

*   **不变法则**——对于任意给定的性质，都存在一些能够保持它不变的转换，以及一些能够改变它的转换。
*   **不变法则**——对于任意给定的转换，都会保持一些性质不变，同时改变另一些性质。
*   **不变法则**——要理解变化，只有通过观察那些保持不变的东西；要理解恒久，只有通过观察那些发生变化的东西。

## 4. 分割

数学上的精确分类（遵守反射性、对称性和传递性）与科学的实用分类之间，存在着很大的区别。

*   **反射条件**——对于每个状态 x，有序对 (x, x) 必须存在于关系表中。
*   **对称性**——d 与 h 是否相同？h 与 d 是否相同？对于这两个问题，应该有相同的答案。
*   **传递性**——如果 B 是 A 的朋友，C 是 B 的朋友，那么 C 一定是 A 的朋友。很明显，对于朋友关系而言，传递性并不成立。

此外，由于整体大于部分之和，因此分解总是会损失一部分整体的特征。这意味着真实系统的特性是无法被完全研究的。

## 5. 强连接定律

容易被分解的“松散组织”早已被分解殆尽，剩下的都是“系统”这种难啃的硬骨头。这些系统的内部连接紧密程度高于平均水平，其中任何一部分的改变，都会对整个系统的性质产生影响。

*   **强连接定律**——平均而言，系统的连接紧密程度要高于平均水平。
*   **强连接定律**——系统由多个部分组成，其中任何一部分都无法被单独改变。
*   **强连接定律**——在系统中，很少有事物是完全等同的。

# 对行为的描述

## 1. 仿真——白箱

与解释和分解观察结果的“黑箱”方法相对的，是“白箱”方法，即对系统进行仿真。

## 2. 状态空间

由于人类的视觉感知仅限于三维空间，因此仿真的“状态空间”通常被限制在二维或三维（即两种或三种属性）中。此时，空间中的一个点就代表了系统的一种状态。如果维度高于三维，我们通常需要采用投影等方式对其进行降维，但这无疑会丢失部分信息。因此，面对投影图像时，说“这是立方体的图像”要比直接说“这是立方体”更为精确。

*   **图像法则**——在谈到与维度降低相关的问题时，无论你想表达什么，都请在最后加上“……的图像”这几个字。
*   **历时法则**——如果行为线发生自相交叉的现象，那么该系统要么不是由状态决定的，要么就是我们看到的只是一个投影，而非完整的视图。
*   **共时法则**——如果在同一时刻，两个系统处于状态空间中的同一位置，那么很可能是因为空间维度过低，导致视图不够完整。

除了使用投影的方法进行降维外，我们还可以采用视点变换的方法：将多个属性结合成为数量较少的属性，并在其中保留每个属性的一小部分，而不是像投影方法那样完全放弃某些属性。

可以看出，一旦某个属性“浮出水面”，白箱方法就能够将其“来源”分析得十分透彻。但这里存在一个前提，即如果不经过适当的变换就对行为进行观察，我们可能根本无法提取出合适的属性。

## 6. 旧车定律

*   **旧车定律**——调节作用发挥良好的系统，不需要进行适应性变化。系统可以通过其适应性变化来简化其调节工作。
*   **旧车定律**——如果一种看待世界的方法没有对观察者产生压力，那么它就不需要改变。如果一种看待世界的方法对观察者产生了压力，那么就需要对其进行改变。
